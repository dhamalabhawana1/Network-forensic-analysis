# Network Forensic Analysis Using NetworkMiner

## Overview
This project demonstrates network forensic investigation conducted in a controlled cybersecurity lab environment using NetworkMiner.

The project focused on analyzing captured network traffic, reconstructing sessions, and examining network communication behavior from packet capture (PCAP) files.

---

## Tool Used
- NetworkMiner
- Wireshark (basic traffic verification)

---

## Objectives
- Analyze captured network traffic
- Investigate communication behavior
- Reconstruct network sessions
- Examine extracted network artifacts
- Understand packet-level forensic investigation

---

## Activities Performed
- Imported and analyzed PCAP files using NetworkMiner
- Investigated transmitted network data
- Reconstructed sessions and extracted artifacts
- Examined communication behavior between systems
- Performed basic traffic verification using Wireshark

---

## Key Findings
- Observed network communication patterns
- Examined transmitted artifacts from captured traffic
- Investigated suspicious network behavior within the lab environment
